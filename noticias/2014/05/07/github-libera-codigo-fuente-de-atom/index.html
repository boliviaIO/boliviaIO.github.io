Github Libera Código Fuente de Atom

La empresa Github anunció ayer 6 de Mayo la liberación del código fuente de su último proyecto más grande Atom, un editor para desarrolladores, al mismo tiempo puso disponible para descarga una beta pública para los usuarios OSX anunciando su disponibilidad para Linux y Windows en pocas semanas.
